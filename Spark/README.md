###### Installing Spark and IPython Notebook    
Get Docker  
docker run -d -t -p 8888:8888 prabeeshk/pyspark-notebook  
Use your web browser with http://127.0.0.1:8888 or http://localhost:8888  
Check if your container is running
```
docker ps
```
